Gender: This field indicates the gender of the student, with values "male" and "female."

Race/Ethnicity: This field specifies the race or ethnicity group to which the student belongs.

Parental Level of Education: This field represents the highest level of education achieved by the student's parents. 

Lunch: This field indicates the type of lunch the student receives, with values "standard" or "free/reduced"

Test Preparation Course: This field describes whether the student completed a test preparation course. Values include "completed" and "none."

Math Score: This field contains the student's score in the math subject.

Reading Score: This field contains the student's score in the reading subject.

Writing Score: This field contains the student's score in the writing subject.
